# IPL Win Prediction

This project predicts the winning probability of a team during an IPL match using machine learning techniques. 
The model analyzes historical match data and match conditions to estimate the chances of each team winning the match.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## Dataset
The dataset contains historical matches from the 
Indian Premier League (IPL), including information such as teams, venue, target score, runs remaining, wickets remaining, and overs.

## Features Used
- Batting Team
- Bowling Team
- Target Score
- Runs Remaining
- Balls Remaining
- Wickets Remaining
- Current Run Rate
- Required Run Rate

## Project Workflow
1. Data Collection and Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Model Training
5. Prediction of Match Winning Probability

## Result
The trained machine learning model predicts the probability of a team winning the match based on real-time match conditions.
